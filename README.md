# 🛠 Моя Цифровая Мастерская (My Workshop)

Добро пожаловать в мой личный репозиторий полезных скриптов для автоматизации Linux (MX Linux / Debian).

## 🚀 Основные инструменты

В папке `scripts/` вы найдете решения для самых разных задач:
* **Системные:** обновление системы (`apt-update-upgrade.sh`), настройка Wine.
* **Файловые:** резервное копирование (`copy_of_section_home.sh`), создание ярлыков.
* **Организация:** автоматический каталогизатор ваших файлов.

---

### 📦 Как пользоваться каталогом
Для удобства навигации в корне репозитория есть файл [files.txt](./files.txt). 
Он содержит прямые ссылки на скачивание каждого скрипта. 

| Скрипт | Описание |
| :--- | :--- |
| `apt-update-upgrade.sh` | Полное обновление системы одной командой |
| `auto-shortcut.sh` | Быстрое создание ярлыков на рабочем столе |
| `Универсальный_Wine` | Скрипт для быстрой настройки Wine окружения |

---

### ⚙️ Автоматизация загрузки
Проект поддерживается с помощью системы **Uploader**:
1. Новые скрипты попадают в папку `staging`.
2. Автоматически индексируются в `files.txt`.
3. Синхронизируются с GitHub через `upload.sh`.
